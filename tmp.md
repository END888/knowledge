

| **句型类型**       | **结构**                | **含义**          | **例句**                                  | **注意事项**                                                                 |
|--------------------|-------------------------|-------------------|-------------------------------------------|-----------------------------------------------------------------------------|
| **肯定句**         | 名词1 は 名词2 です      | A是B              | 李さんは中国人です。<br>（小李是中国人。）  | ①「は」读作「わ」；<br>②姓氏后直接加「です」，无需「さん」（例：王です）。        |
| **疑问句**         | 名词1 は 名词2 ですか     | A是B吗？          | キムさんは中国人ですか？<br>（金女士是中国人吗？） | ①句尾加「か」表疑问；<br>②回答方式：<br>肯定→はい，そうです<br>否定→いいえ，ちがいます |
| **否定句**         | 名词1 は 名词2 では ありません<br>（口语：じゃ ありません） | A不是B           | 森さんは学生では ありません。<br>（森先生不是学生。） | ①「では」可简化为「じゃ」，语气更随意；<br>②口语常用「じゃ ありません」。          |
| **存在位置句**     | 名词1 は 名词2（场所）です | A在某地           | 食堂はデパートの7階です。<br>（食堂在百货商店7楼。） | ①「です」表示存在位置，非“是”；<br>②区分：<br>・トイレはここです（厕所在这儿）<br>・ここはトイレです（这里是厕所）。 |




| **关联词类型**     | **日语结构**                  | **含义**          | **例句**                                                                 | **注意事项**                                                                 |
|---------------------|-------------------------------|-------------------|--------------------------------------------------------------------------|-----------------------------------------------------------------------------|
| **递进关系**        | ～だけでなく～も<br>（～ばかりでなく～も） | 不但…还…          | 彼は**頭がいいだけでなく**、**努力もする**。<br>（他不但聪明，还很努力。）     | ①后句常用「も」呼应；<br>②「ばかりでなく」更书面化。                             |
| **转折关系**        | ～が、～<br>～けれども、～    | 虽然…但是…        | **雨が降っているが**、**散歩に行く**。<br>（虽然下雨，但还是去散步。）         | ①「が」多用于书面；<br>②「けれども」口语中可简化为「けど」。                       |
| **并列关系**        | ～も～し、～も～              | 既…又…            | 彼は**歌も上手だし**、**ダンスも上手だ**。<br>（他既会唱歌，又会跳舞。）        | ①「し」连接并列分句；<br>②前后结构需对称。                                       |
| **因果关系**        | ～ので、～<br>～から、～      | 因为…所以…        | **雨が降ったので**、**試合は中止になった**。<br>（因为下雨，所以比赛取消了。）   | ①「ので」表客观原因；<br>②「から」表主观原因（例：疲れたから休む）。               |
| **条件关系**        | ～ば、～<br>～たら、～        | 如果…就…          | **練習すれば**、**上手になる**。<br>（如果练习，就会变好。）                     | ①「ば」用于一般条件；<br>②「たら」强调假设结果（例：時間があったら行く）。          |
| **让步关系**        | ～ても、～<br>～のに、～      | 即使…也…<br>明明…却… | **雨が降っても**、**出かける**。<br>（即使下雨，也要出门。）                     | ①「ても」表逆接条件；<br>②「のに」含意外或抱怨语气（例：頑張ったのに失敗した）。    |
| **选择关系**        | ～か～か                      | 要么…要么…        | **コーヒーか**、**紅茶か**、どちらにしますか？<br>（要咖啡还是红茶？）           | ①多用于疑问句；<br>②需与「どちら」或「どれ」搭配使用。                            |

|                   | **第二人称**       | あなた、きみ、おまえ       | 你                 | あなたの名前は？                          | 你的名字是？                     | ①「おまえ」粗鲁，慎用；<br>②对上级用「○○さん」代替代词。                      |
|                   | **第三人称**       | 彼（かれ）、彼女（かのじょ）| 他、她             | 彼女は医者です。                          | 她是医生。                       | ①「彼女」可指女朋友；<br>②口语中常用「あの人」代替。                          |
|                   | **复数**           | わたしたち、ぼくたち       | 我们               | わたしたちは友達です。                    | 我们是朋友。                     | 复数后缀「たち」或「ら」（例：彼ら）。                                        |
| **指示代词**      | **近指**           | これ、ここ、この           | 这、这里、这个      | これは本です。                            | 这是书。                         | ①「これ」指说话者附近；<br>②「ここ」表具体位置。                               |
|                   | **中指**           | それ、そこ、その           | 那（对方附近）、那个| そのペンをください。                      | 请给我那支笔。                   | ①「それ」指听话者附近；<br>②「そこ」表听话者附近的位置。                       |
|                   | **远指**           | あれ、あそこ、あの         | 那（远处）、那个    | あの建物は学校です。                      | 那座建筑是学校。                 | ①「あれ」指双方远处；<br>②「あそこ」表远距离位置。                            |
|                   | **疑问**           | どれ、どこ、どの           | 哪个、哪里、哪个    | トイレはどこですか？                      | 厕所在哪里？                     | 疑问代词需搭配助词（例：どれがいい？）。                                      |
| **疑问代词**      | **人物**           | だれ                       | 谁                 | だれが来ますか？                          | 谁会来？                         | 礼貌场合用「どなた」（例：どなたですか？）。                                  |
|                   | **事物**           | 何（なに/なん）            | 什么               | これは何ですか？                          | 这是什么？                       | 「なん」为口语缩略（例：何時→なんじ）。                                       |
|                   | **时间**           | いつ                       | 什么时候           | 会議はいつ始まりますか？                  | 会议什么时候开始？               | 回答需具体时间（例：午後３時です）。                                          |
|                   | **原因**           | どうして                   | 为什么             | どうして遅れたんですか？                  | 为什么迟到了？                   | 同义词「なぜ」（更正式）。                                                   |
| **不定代词**      | **人物不定**       | 誰か（だれか）             | 某人               | 誰かが窓を開けました。                    | 有人打开了窗户。                 | 否定形式为「誰も～ない」（例：誰もいない）。                                   |
|                   | **事物不定**       | 何か（なにか）             | 某物               | 何か飲み物が欲しいです。                  | 我想喝点什么。                   | 否定形式为「何も～ない」（例：何も要りません）。                               |
|                   | **场所不定**       | どこか                     | 某处               | どこか静かな場所に行きましょう。          | 我们去个安静的地方吧。           | 否定形式为「どこも～ない」（例：どこも開いていない）。                         |
